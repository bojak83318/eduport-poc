# EduPort Video Tutorial Script

**Duration:** 2:30 - 3:00 minutes  
**Production URL:** https://eduport-poc.vercel.app/  
**Demo Wordwall URL:** https://wordwall.net/resource/101727466/esa/have-got-has-got-affirmative-negative

---

## Scene 1: Intro (0:00-0:15)
**Visual:** EduPort Homepage
**Audio:** "Hi, I'm going to show you how to convert Wordwall activities to H5P format using EduPort. It takes less than 30 seconds."

## Scene 2: Find Wordwall URL (0:15-0:30)
**Visual:** Wordwall Community Page → Click on activity → Copy URL
**Audio:** "First, go to Wordwall and find an activity you want to convert. Copy the URL from your browser - it should look like wordwall.net/resource/12345678"
**Demo URL to use:** `https://wordwall.net/resource/101727466/esa/have-got-has-got-affirmative-negative`

## Scene 3: Use EduPort (0:30-1:00)
**Visual:** EduPort Converter Page → Paste URL → Click Convert → Show "Transpiling..."
**Audio:** "Now go to eduport.app. Paste your URL here. Check the box confirming you have permission to use the content. Click Convert."

## Scene 4: Download (1:00-1:15)
**Visual:** Conversion Success → Download button → Browser download
**Audio:** "The conversion takes just a few seconds. Your H5P file will download automatically to your computer."

## Scene 5: Upload to Moodle (1:15-2:30)
**Visual:** Moodle Course Page → Add Activity → H5P → Upload → Save
**Audio:** "Now let's upload to Moodle. Go to your course, turn editing on, add an activity - choose H5P, upload the file you just downloaded, and save."

## Scene 6: Test the Activity (2:30-2:50)
**Visual:** Moodle Activity View → Interactive Quiz
**Audio:** "Click on the activity to test it. As you can see, the quiz works just like it did on Wordwall, but now it's embedded in your Moodle course."

## Scene 7: Outro (2:50-3:00)
**Visual:** EduPort Homepage with URL
**Audio:** "That's it! Convert any Wordwall activity to H5P in seconds. Visit eduport.app to try it free."
